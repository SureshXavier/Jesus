Host Discovery using Lua Script

This Lua script is designed to scan a network for active hosts and report their IP addresses. It uses the Lua programming language to perform the scan. The script sends ICMP ECHO_REQUEST packets to potential hosts and waits for a response. If a response is received, the script reports the IP address of the responding host.

Usage of this script

To use this script, you will need to have the Lua programming language and the LuaSocket library installed on your system. Once you have these dependencies installed, download the script and save it to your local directory. You can then run the script from the command line using the Lua interpreter.
